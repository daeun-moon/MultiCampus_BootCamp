1st 분석 : 판매랭킹에 올라와 있는 브랜드들의 빈도 비교. 
-> 크롤링 후 박스플랏, 바플랏 그리기. 워드클라우드 까지 만들어 확인
=> 각 브랜드 판매자는 올리브영 판매랭킹 차지 1위 브랜드라고 광고 할 수 있다. 

2nd-1 분석 : 상관관계 확인 : 순위와 옵션과의 관계
-> 점 양분 상관계수를 활용 하여 확인 : (크롤링하는 시각에 따라 결과가 달라지겠지만....)
-> 결과해석 : 순위에서만 유일하게 pvalue가 0.09로 0.1보다 작았다. 유의수준을 0.1로 하여 신뢰도 90%로 했을때 유의수준보다 작으므로 귀무가설을 기각 할 수 있게 된다. 즉, 두 대상은 상관관계가 있다고 볼 수 있다.  
(귀무가설 : 두 대상은 상관관계가 없다)
=> 브랜드 판매자의 입장에서는 판매순위를 올리기 위해서는 증정품 추가를 고민해보는 것도 긍정적

2nd-2 분석 : 상관관계 확인 : 브랜드랭크빈도와 옵션과의 관계
위와 동일하게 진행( 순위랑 옵션이 상관이 있었으면...브랜드랭크빈도와도....?)
-> 결과 해석 : pvalue가 모두다 0.1보다 훨씬 크게 나왔다. 즉, 귀무가설 채택! 상관관계 없다!!!! 

+) 브랜드랭크빈도와 랭크는 전혀 상관이 없다.....

2nd-3 분석 : 현재 판매가 또는 정가와 랭킹차지 빈도와의 관계
-> 현재판매가와 빈도와의 상관관계 없음. 정가와 빈도와의 상관관계도 없음.(정가와 현재판매가와는 상관관계 존재)
=> 가격이 저렴한 브랜드일수록 랭킹을 더 많이 차지 할 것이라고 생각했다. But상관계수를 확인해보면 거의 0에 가까우므로 이는 상관관계가 없다라고 말할 수 있다. 그럼 나의 직관은 틀린것인가? SO 이번에는 현재판매가와 정가 그리고 순위자체를 파악하기로!!!!

2nd-4 분석
=> 랭크와 현재가, 랭크와 정가 모두 -0.25정도의 상관계수를 보였다. 절댓값 기준 0.2 ~ 0.4를 낮은 상관관계라고 칭한다. 현재가가 높을수록 랭크가 낮아지는 관계가 아주 낮게 보이고, 정가가 높을수록 랭크가 낮아지는 관계가 아주 낮게 보인다고 해석할 수 있다. 
  처음에는가격이 저렴할 수록 높은 순위에 위치할 것이라고 생각했다. 하지만 아주 낮은 상관관계를 보이는 것으로 보아 올리브영 랭킹시장은 가격만 중요시 되는 시장은 아니다. 이는 다양한 연령층이 사용하는 시장이기 때문이라고 생각한다.
